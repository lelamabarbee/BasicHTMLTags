Borders

The border property allows you to specify the style, width, and color of an element's border.

Border Styple

The border-style property can take up to four values (top border, right border, bottom border, left border). The following border styles are allowed:

  • dotted: dotted border
  • dashed: dashed border
  • solid: solid border
  • double: double border
  • groove: 3D groove border, depends on border-color value
  • ridge: 3D ridged border, depends on border-color value
  • insert: 3D insert border, depends on border-color value
  • none: no border at all
  • hidden: hidden border
  • mixed: you can set a different border style for the top, bottom, right and left
